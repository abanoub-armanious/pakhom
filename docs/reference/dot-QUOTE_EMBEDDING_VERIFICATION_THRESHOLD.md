# Embedding cosine similarity threshold for the verification ladder's step 4 Justified at 0.85 because thematic-analysis paraphrases that retain meaning (as opposed to fabrications) cluster above this threshold in pilot tests. Lower thresholds admit too many fabrications (false positives); higher thresholds reject legitimate paraphrases (false negatives).

Embedding cosine similarity threshold for the verification ladder's step
4 Justified at 0.85 because thematic-analysis paraphrases that retain
meaning (as opposed to fabrications) cluster above this threshold in
pilot tests. Lower thresholds admit too many fabrications (false
positives); higher thresholds reject legitimate paraphrases (false
negatives).

## Usage

``` r
.QUOTE_EMBEDDING_VERIFICATION_THRESHOLD
```

## Format

An object of class `numeric` of length 1.
