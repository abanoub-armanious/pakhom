# Send a quick completion using the fast/cheap model

Send a quick completion using the fast/cheap model

## Usage

``` r
ai_complete_fast(
  provider,
  prompt,
  system_prompt = NULL,
  task = "sentiment",
  json_mode = FALSE
)
```

## Arguments

- provider:

  AIProvider object

- prompt:

  User prompt

- system_prompt:

  System prompt

- task:

  Task name

- json_mode:

  JSON mode

## Value

Character string response
