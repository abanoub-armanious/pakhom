# Hydrate a Code S3 from a coding_state codebook entry

Hydrate a Code S3 from a coding_state codebook entry

## Usage

``` r
.code_from_codebook(key, coding_state)
```

## Arguments

- key:

  Codebook key

- coding_state:

  ProgressiveCodingState

## Value

Code S3 (stub if key not found in codebook)
