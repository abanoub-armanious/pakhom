# Aggregate overall analysis statistics for report

Aggregate overall analysis statistics for report

## Usage

``` r
aggregate_overall_statistics(
  data,
  theme_set,
  consolidated = NULL,
  learning_context = NULL,
  config = NULL
)
```

## Arguments

- data:

  tibble with all analysis columns

- theme_set:

  ThemeSet object

- consolidated:

  ConsolidatedCodes list (or NULL)

- learning_context:

  LearningContext object (or NULL)

- config:

  ThematicConfig object (or NULL)

## Value

List of overall statistics
