# Export CSV files for each theme's entries

Export CSV files for each theme's entries

## Usage

``` r
export_theme_entry_csvs(data, theme_set, output_dir)
```

## Arguments

- data:

  tibble with theme_membership\_\* or emerged_themes columns

- theme_set:

  ThemeSet object

- output_dir:

  Output directory

## Value

Named list of file info per theme
