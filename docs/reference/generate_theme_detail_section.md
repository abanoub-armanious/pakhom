# Generate a theme detail section for appendix

Generate a theme detail section for appendix

## Usage

``` r
generate_theme_detail_section(theme_name, ts, theme_index, theme_csv_info)
```

## Arguments

- theme_name:

  Theme name

- ts:

  Theme statistics (from aggregate_theme_statistics)

- theme_index:

  Numeric index

- theme_csv_info:

  CSV file info list (or NULL)

## Value

R Markdown string for the section
