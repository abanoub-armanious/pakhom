# Enrich themes with entry counts, sentiment, and quotes

Enrich themes with entry counts, sentiment, and quotes

## Usage

``` r
enrich_themes(theme_set, data, coding_state = NULL)
```

## Arguments

- theme_set:

  ThemeSet object

- data:

  Tibble with theme_membership\_\* and sentiment columns

- coding_state:

  ProgressiveCodingState (optional)

## Value

Enriched ThemeSet
